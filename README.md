# 📊 CRM (Customer Relationship Management) System

A full-stack CRM application designed to manage customer information, track interactions, and organize business operations efficiently.

This project demonstrates frontend, backend, database integration, and API development while building a real-world web application.

## 🚀 About The Project

The CRM system helps businesses manage customer data, monitor customer relationships, and maintain organized records through an easy-to-use interface.

## ✨ Features

- User authentication
- Customer management
- Add, update, delete customer records
- Search and filter customers
- Dashboard overview
- REST API integration
- Responsive user interface
- Database management
